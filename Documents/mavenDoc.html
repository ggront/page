How to install Java Development Kit and Apache-Maven on Mac OS X

Purpose:

The primary purpose of installing and configuring these softwares is to enable automated testing via the Selenium Safari WebDriver to run automated tests on the Safari web browser. Given that Safari is only available for Mac OS X and is a popular browser for users, automated testing will be necessary for scaling any departments automated testing capabilities

Approach:

First you should check if your machine currently has any of the require softwares installed.

  • To check the JDK version, run 'java --version' in your machines terminal.
  • To check the Maven version, run 'mvn -v' in your machines terminal.
  • To check the GIT version, run 'git --version' in your machines terminal.

    • If an error was thrown for any of the required software, you will need to install the latest version of those softwares

    Download Links:

  • Java Development Kit
  • Maven-Apache
  • Git

    • Environment Setup (Mac OS X):

      Installing JDK:

      1. Install the x64 DMG Installer Java Developer Kit (JDK) from the download link above or here.
      2. Open the DMG installer (this file should end in .dmg) by double clicking on the file.
      3. Double click on the Package (this should end in .pkg) to start the installation process.
      4. Click Continue then Install. You may have to enter your password to proceed. If you do not have sudo permissions, you will need to be granted access. Click Install Software.
      5. If you open a Terminal session and type java -version you should see the current version of java on your machine. For further help refer to Oracles' Documentation here.
      6. If you type echo $JAVA_HOME in Terminal, you should see the following response:

          /Library/Java/JavaVirtualMachines/(jdk-version).jdk/Contents/Home

        1. If this is not the response you receive, troubleshoot to find where your Java_Home directory is located and move it into the Contents directory located below:

            2. /Library/Java/JavaVirtualMachines/(jdk-version).jdk/Contents

      7. Open your bash profile by typing:

          vim ~./bash_profile

      8. Add the following text to the top (Make sure you save this change):

          export JAVA_HOME=/Library/Java/JavaVirtualMachines/(The jdk file you downloaded)/Contents/Home

      Installing Maven:

      1. Download the ‘Binary tar.gz archive’ Apache-Maven file from the download link above or here.

        1. In Terminal navigate to where you want the Apache-Maven directory to be created and type the following:

            unzip (entire file name). An example of this is: unzip apache-maven-3.8.7-bin.tar.gz

        2. You may also double-click on the file after moving it to your desktop or in your Downloads folder.

      2. You should now see the Apache-Maven directory. You can check this by typing:

          ls

        • NOTE: Now, you should see a directory created called apache-maven-(Version you downloaded).

      3. Reopen your bash profile by typing the snippet below. Make sure you are in the in the same User directory as before.

          vim ~./bash_profile

      4. Now we need to add the bin directory of the created directory to the PATH environment variables. Add the following code and make sure your bash profile matches (Make sure you save this change):

          export JAVA_HOME=/Library/Java/JavaVirtualMachines/(The jdk file you downloaded)/Contents/Home
          export MAVEN_HOME=/Users/(Your user)/(Version you downloaded)
          export PATH=$PATH:$MAVEN_HOME/bin

      5. Close your current Terminal session and reopen a new Terminal session.
      6. Type the following:

          source .bash_profile

      7. Now Apache-Maven should be installed and you can check the status of the service by typing:

          mvn -v

      NOTE: Upon closing your terminal session and reopening a new Terminal session, you will have to run source .bash_profile every time to start Apache-Maven. If this is too much work, you may set a startup command in Terminal by following the steps below:

      1. Open Terminal
      2. Visit ‘Terminal’ > ‘Settings’ in the top menu bar.
      3. Select ‘Profiles’
      4. Select ‘Shell’
      5. Select the checkbox next to ‘Run command:' and type source .bash_profile in the input field.
      6. Close and reopen Terminal.
